# Sistema de Controle de Biblioteca

## Objetivo
Este projeto tem como objetivo auxiliar no gerenciamento de livros, empréstimos e devoluções de uma biblioteca escolar.

## Funcionalidades
- Cadastro de livros
- Cadastro de alunos
- Controle de empréstimos
- Controle de devoluções

## Tecnologias
- Git
- GitHub
